tS Desktop builds

Automated builds for translationStudio desktop can be found at the below links.

  • develop.ts-desktop.surge.sh develop branch
  • release.ts-desktop.surge.sh release branches
  • sandbox.ts-desktop.surge.sh all other branches

Builds will only trigger if all tests pass.

Builds are are not triggered on pull requests.

Builds will not be created for the master branch.

Builds are generated for tags but they are instead uploaded directly to releases on github.
